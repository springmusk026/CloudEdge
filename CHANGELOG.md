# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-09

### Added
- Full Hono.js Worker API (auth, posts, tags, media, comments, newsletter, AI, webhooks)
- 4 Durable Objects (PostCollaboration, LiveComments, Analytics, BuildQueue)
- D1 database schema with 22+ tables, Drizzle ORM, migrations
- Admin SPA with shadcn/ui, TanStack Query, Tiptap rich text editor
- Command palette (⌘K) for quick navigation
- WordPress-style site customizer with live preview
- Public frontend with React, Framer Motion, reading progress, TOC, bookmarks
- Queue workers for image processing and email sending (Resend API)
- Post scheduling with cron-based auto-publish
- Draft sharing via secret preview links
- Content import from URL
- Announcement banner system
- Reading list / bookmarks (localStorage)
- Membership paywall rendering
- SEO meta tags + JSON-LD structured data
- Turnstile CAPTCHA component
- GitHub Actions CI/CD pipeline
- RSS, JSON Feed, XML Sitemap generation
